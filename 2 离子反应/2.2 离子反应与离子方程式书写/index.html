    2.2 离子反应与离子方程式书写¶

    A.离子反应及其表示方法¶

    a.离子反应定义 有离子参加或生成的反应。
    b.离子方程式 用实际参加反应的离子符号来表示反应的式子。
      e.g. \(\ce{HCl + NaOH =\!= NaCl + H2O}\)中，\(\ce{Cl-}\)和\(\ce{Na+}\)实际上没有参加反应(旁观离子），因此该反应的离子方程式写成\(\ce{H+ + OH- =\!= H2O}\)。

    注意

    注意区分电离方程式和离子方程式：电离方程式指的是电离（物理变化）的过程，而离子方程式指点是化学反应的过程。

    c.离子方程式书写步骤
      以\(\ce{AgNO3、NaCl}\)反应为例：
      1.写 写出化学方程式 \(\ce{AgNO3 + NaCl =\!= AgCl v + NaNO3}\)
      2.拆 把易溶于水且易电离的物质写成离子形式，难溶的物质、气体、水仍然用化学式表示。 \(\ce{Ag+ + NO3- + Na+ + Cl- =\!= AgCl v + Na+ + NO3-}\)
      3.删 删去两边不参加反应的离子（旁观离子），并且化为最简。\(\ce{Ag+ + Cl- =\!= AgCl v}\)
      4.查 检查两边原子个数和电荷总数是否相等。

    注意

    1.在水溶液中完全自由移动的离子形式存在的物质才可以拆成离子的形式。
    2.浓硫酸不拆成离子。
    3.强碱\(\ce{Ca(OH)2}\)是微溶性物质，微溶物作为产物一律不拆；作为反应物，浑浊不拆，澄清拆。
    4.易溶的强电解质盐，如果产物以沉淀形式析出（溶解不了），则不拆。e.g. 侯氏制碱法： \(\ce{NaCl + H2O + NH3 + CO2 =\!= NH4Cl + NaHCO3 v}\)的离子方程式就是\(\ce{Na+ + H2O + NH3 + CO2 =\!= NH4+ + NaHCO3 v}\)。
    5.氧化物一律不拆。

    d.离子方程式意义 可以用于表示同一类型的离子反应，比如\(\ce{H+ + OH- =\!= H2O}\)可以用于表示酸碱中和反应。
    e.复分解型离子反应发生的条件
      1.溶解度大的盐制取溶解度小的盐。（相当于初中的产生沉淀）
      e.g. 氯化钙溶液和氢氧化钠溶液反应：\(\ce{Ca^2+ + 2OH- =\!= Ca(OH)2 v}\)
      2.难挥发的（沸点高）制取易挥发的（沸点低）。（相当于初中的产生气体/前面提到的强酸制弱酸）
      e.g.稀硫酸与碳酸钠溶液反应：\(\ce{2H+ + CO3^2- =\!= CO2 ^ + H2O}\)
      3.强电解质制弱电解质
      e.g. 醋酸钠和盐酸反应：\(\ce{CH3COO- + H+ = CH3COOH}\)
      注：上述反应三者满足一个即可，有时候甚至会违背其中的一些规律。
      e.g. \(\ce{H2S + CuSO4 =\!= CuS v + H2SO4}\) 违背强酸制弱酸，但是遵循“溶解度大制溶解度小”，因此能够进行。

    B.与量有关的离子反应方程式¶

    a.基础规则技巧
      1. 少量视为“1”，过量者要多少有多少（配合少量）
      2. 先写中和，后写沉淀
      3. 少量者符合化学式比例，过量者不一定
      4. 优先级判定：i.生成难溶物的优先于生成微溶物的；ii.剩余的优先级用”反证法“判断。
    b.实例
      \(\ce{NaHCO3}\)和\(\ce{Ca(OH)2}\)反应：
      e.g.1 \(\ce{NaHCO3}\)少量
      step0 分离离子：\(\ce{Na+}\) \(\ce{HCO3-}\) \(\ce{Ca^2+}\) \(\ce{OH-}\) 并且确认其中钠离子不参与反应。
      step1 将\(\ce{HCO3-}\)系数视作“1”：\(\ce{HCO3-}\)（\(\ce{Na+}\)不反应，因此不考虑）
      step2 确定\(\ce{OH-}\)的量：一个\(\ce{HCO3-}\)只能和一个\(\ce{OH-}\)反应，因此\(\ce{OH-}\)系数为1：\(\ce{HCO3- + OH-}\)
      step3 确定\(\ce{Ca^2+}\)发现上述反应生成\(\ce{CO3^2-}\)，发现一个\(\ce{Ca^2+}\)离子足够与其反应。因此得到：\(\ce{HCO3- + OH- + Ca^2+ =\!= CaCO3 v + H2O}\)

      e.g.2 \(\ce{Ca(OH)2}\)少量
      step0 同上，拆分离子，得出钠离子不反应。
      step1 少量者的整体系数设置为1（也就是，\(\ce{Ca(OH)2}\)的系数设为1，对应1个钙离子和2个氢氧根离子）：\(\ce{Ca^2+ + 2OH-}\)
      step2 确定\(\ce{HCO3-}\)的量：用氢氧根离子的量来确认，2个\(\ce{HCO3-}\)足以和2个氢氧根离子完全反应，因此将\(\ce{HCO3-}\)的系数设为2，并且得出最终结果：\(\ce{Ca^2+ + 2OH- + 2HCO3- =\!= CaCO3 v + 2H2O + CO3^2-}\)

    注意

    当两种物质有相同离子的时候，需要对两种物质中的相同的离子分开分析，不能简单的按照上面的规律来。比如少量\(\ce{Ca(HCO3)2}\)和\(\ce{Ca(OH)2}\)反应的离子方程式在约分化简前是\(\ce{2Ca^2+ + 2HCO3- + 2OH- =\!= 2CaCO3 v + 2H2O}\)。这个方程式中，两个钙离子，一个是由系数设为1的\(\ce{Ca(HCO3)2}\)提供的，还有一个是由\(\ce{Ca(OH)2}\)提供的。（但是实际考试需要把所有系数同除以2，化到最简）

    C.离子方程式正误判断¶

    a.看离子反应是否符合客观事实
      e.g. 1.稀硫酸与铁屑反应，生成物中写\(\ce{Fe^3+}\)，明显错误
      2.写钠和硫酸铜发生置换反应，明显错误
      3.反应条件是加热，但生成物中直接写\(\ce{NH3.H2O}\)，明显错误（条件为加热时的正确写法是\(\ce{NH3 ^ + H2O}\)）
      4.写根本不反应的方程式，比如向\(\ce{CaCl2}\)中通入\(\ce{CO2}\)。
    b.看左右两边的原子和电荷是否守恒
      e.g. \(\ce{Fe^2+ + Cl2 =\!= Fe^3+ + 2Cl-}\)左边为+2，右边为+1，不守恒，因此错误。

    注意

    不要忘记了离子前面的系数。

    c.看是否符合拆写规则（重要）
      这一步的主要关注点就是：弱电解质不能拆，强电解质必须拆。
      e.g. 醋酸去水垢，如果方程前面反应物写的是\(\ce{H+}\)而不是\(\ce{CH3COOH}\)，那么说明出错了。（醋酸是弱电解质，不能拆开）
      此外，有些电解质有时候并没有真正电解（比如很多不溶性碱）。也就是说，沉淀不能拆。比如，如果将\(\ce{Cu(OH)2}\)拆写成\(\ce{OH-}\)，就是错的。只有真正完全电离了的物质才可以拆，比如活泼金属氧化物（如氧化钙）也属于强电解质，但在水中会迅速发生反应，没有电离，因此通常活泼金属氧化物也不能拆开。
      以及，有一些方程式会出现拆解不完全的情况，也是错的。
    d.看方程式中是否有遗漏的离子反应
      e.g. \(\ce{Ba(OH)2}\)和\(\ce{CuSO4}\)反应时，四种离子均发生了反应，如果只写了两个离子的就错了。
    e.看反应物或产物配比是否正确
      e.g. 一个\(\ce{Ba(OH)2}\)作为反应物的反应，离子方程式中却只写了\(\ce{Ba^2+ + OH-}\)，系数配比不正确，因此错误。
      此外，当题目中有至溶液呈中性、至恰好沉淀完全之类的限定词时，还要考虑符合题目条件时反应物的配比。
      e.g.\(\ce{NaHSO4}\)和\(\ce{Ba(OH)2}\)反应至中性，则需要让两者系数的比例为2:1。如果发现题目给出的不符合这一比例，则说明错误。
      再比如，同样是上面的两个反应物，但是限定条件改成“恰好沉淀完全”，那么就需要让两者的系数为1:1。由此可见，同样的反应物，不同的限定条件也肯能会让比例不同。
    f.看是否符合题目设定条件 根据上述的B.与量有关的离子反应方程式判断。
