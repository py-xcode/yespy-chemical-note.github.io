    1.1 根据物质的组成和性质分类¶

    A.物质的分类¶

    a. 元素、物质和微粒间的关系
      
      同素异形体：由同种元素形成的几种性质不同的单质。e.g. \(\ce{O2}\)、\(\ce{O3}\)。

    注意

    1. 同素异形体性质不同，是因为单质分子结构不同。
    2. 同素异形体相互转化属于化学变化。
    3. 只含有一种元素的物质不一定是纯净物，比如\(\ce{O2}\)、\(\ce{O3}\)的混合物。

    b.常见的分类方法：树状分类法（认识从属关系），交叉分类法（认识一种物质的多重属性）。

    B.氧化物¶

    注意

    下面三种分类都是在成盐氧化物种进行分类的，不成盐氧化物不在下列分类之中。 氧化物中除去酸性氧化物、碱性氧化物和两性氧化物以外，还有不成盐氧化物（\(\ce{CO}\)）、过氧化物（\(\ce{Na2O2}\)）等。

    a.酸性氧化物
      1.定义：能和碱反应生成盐和水，且化合价不变的氧化物。
      2.常见物质种类：绝大多数非金属氧化物（不包括不成盐氧化物，例如\(\ce{CO}\)）、部分金属氧化物（例如\(\ce{Mn2O7}\)）。
      3.通性：
      1. 能和碱反应生成盐和水，且化合价不变。
      2. 能和碱性氧化物生成盐，且化合价不变。
      3. 若能和水反应（能溶于水），则生成对应价态的酸。
      注释：
      1，3：注意要求化合价不变。如，\(\ce{3\overset{+4}{N}O2 + H2O==2H\overset{+5}{N}O3 + \overset{+2}{N}O}\),\(\ce{2\overset{+4}{N}O2 + 2NaOH == Na\overset{+5}{N}O3+Na\overset{+3}{N}O2+H2O}\)
      其中\(\ce{N}\)元素的化合价前后不同，因此 \(\ce{NO2}\)不属于酸性氧化物 。
      部分金属氧化物也是酸性氧化物，比如\(\ce{Mn2O7 + 2KOH==2KMnO4 + H2O}\),其中锰全程都是+7价。
      2：酸性氧化物生成盐的例子：\(\ce{CO2+Na2O==Na2CO3}\)。
      3：\(\ce{SiO2}\)难溶与水，它有对应价态的酸硅酸，但是无法通过直接和水反应获得。
      4.两个不一定：非金属氧化物不一定都是酸性氧化物（比如不成盐氧化物），酸性氧化物不一定都是非金属氧化物（例如\(\ce{Mn2O7}\)）
    b.碱性氧化物
      1.定义：能和酸反应生成盐和水，且化合价不变的氧化物。
      2.常见物质种类：绝大多数金属氧化物（不包括上述的\(\ce{Mn2O7}\)，并且也不包括两性氧化物如\(\ce{Al2O3}\)）。
      3.通性：
      1.能和酸反应生成盐和水，且化合价不变。
      2.能和酸性氧化物生成盐，且化合价不变。
      3.若能和水反应（能溶于水），则生成对应价态的碱。
      1，3：注意要求化合价不变。过氧化物（如\(\ce{Na2O2}\)）必然不是碱性氧化物，因为其中氧元素化合价为1价，和酸反应时会改变化合价。
      3：很多碱性氧化物（比如\(\ce{Fe2O3}\)等）不能溶于水，因此它们无法和水直接反应生成碱。
      4.一个一定：碱性氧化物一定是金属氧化物。
      一个不一定：金属氧化物不一定是碱性氧化物。
    c.两性氧化物
      1.定义：既能和酸反应转化为盐，也能和碱反应转化为盐，且化合价均不改变的氧化物。
      2.常见物质种类：\(\ce{Al2O3}\)、\(\ce{ZnO}\)、\(\ce{Ga2O3}\)、\(\ce{BeO}\)等。
      3.通性：酸性氧化物和碱性氧化物通信的总和。
      e.g.
      \(\ce{Al2O3 + 6HCl == 2AlCl3 + 3H2O}\)
      \(\ce{Al2O3 + 2NaOH + 3H2O == 2Na[Al(OH)4]}\)
      注：
      1.\(\ce{SiO2}\)能够和\(\ce{HF}\)反应，同时能和碱反应，但是其不是两性氧化物，因为\(\ce{SiO2}\)与\(\ce{HF}\)反应得到的\(\ce{SiF4}\)不是盐。
